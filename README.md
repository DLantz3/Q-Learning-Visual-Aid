# Q-Learning Visual

Using numpy and pygame to create a visual representation with Q-Learning.

---

## Setup

Make sure you have python and pygame installed on your machine. Some versions of python may no work with this spcific iteration, for guarenteed results use python 3.13.

---

## Running the game

- Download mouse_maze_qlearning.py
- For MacOS, run python3 mouse_maze_qlearning.py
- For Windows, run python mouse_maze_qlearning.py
- First, click any square to place the cheese
- Run simulations to see how the confidence of each square changes after each training cycle
- Then run the mouse through in real time to see how the pathing is decided, the darker blue the square is, the higher the confidence interval
- Move the cheese by clicking any square and repeat the simulations

---

## Keybinds

---

SPACE: Run 100 training episodes on the maze

R: Run the mouse through the maze in real time

ESC: Exit program
